## Week 1 - History
- One writing system can be used by many languages
- The Epic of Gilgamesh is one of the earliest known major works of literature

## Week 2 - Bits and Bytes
- A bit is a 0 or 1; a byte is 8 bits
- ASCII is a character encoding standard for text

## Week 3 - Operating Systems / Files
- File paths can be absolute or relative
- The shell/terminal lets you run commands and navigate folders

## Week 4 - Git & GitHub
- A commit is a saved snapshot of changes with a message
- GitHub Pages builds a website from Markdown in your repo

## Week 5 - Markdown & Documentation
- Markdown uses # for headings and - for bullet lists
- Links look like: [text](url)
